## Instalación de Laravel 10 con Jetstream y Stack Inertia (Vue 3)



### Requisitos Previos

- "php": "^8.1",
- "Composer": "^2.3,
- "Node"

### instalación

- Clonar Repo
- Configurar la base de datos .env
- composer install
- npm install
- php artisan migrate:refresh -seed
- php artisan optimize
